# IDS Lab - Snort & Suricata

A hands-on Intrusion Detection System lab built using Snort and Suricata on Ubuntu, with Kali Linux as the attack machine.

## Tools Used
- Snort 2.9.20
- Suricata 7.0.3
- Kali Linux (attacker)
- Ubuntu 24.04 (defender)
- VirtualBox

## Attacks Detected
- ICMP Ping sweep
- Nmap SYN port scan
- SSH connection attempts
